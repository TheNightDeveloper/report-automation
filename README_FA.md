# کارنامه الکترونیکی - آموزشگاه شنا

سیستم مدیریت و ایجاد کارنامه الکترونیکی برای آموزشگاه‌های شنا

## ویژگی‌ها

✅ **مدیریت دانش‌آموزان**
- Import لیست دانش‌آموزان از فایل Excel
- نمایش آمار و پیشرفت
- جستجو و مدیریت

✅ **کارنامه الکترونیکی**
- 5 بخش: پیش دبستانی، سال 1-4
- 45 تکنیک شنا
- ارزیابی سه سطحی: عالی، خوب، متوسط
- ذخیره خودکار

✅ **خروجی فایل**
- Export به PDF (قابل چاپ)
- Export به Excel (قابل ویرایش)
- Export تکی یا دسته‌جمعی

✅ **رابط کاربری**
- طراحی مینیمال و زیبا
- پشتیبانی کامل از فارسی
- Responsive (Desktop & Web)
- Keyboard Shortcuts

## نصب و راه‌اندازی

### پیش‌نیازها
- Flutter SDK 3.9.2 یا بالاتر
- Dart SDK

### مراحل نصب

1. Clone کردن پروژه:
```bash
git clone [repository-url]
cd karnameh
```

2. نصب dependencies:
```bash
flutter pub get
```

3. اجرای code generation برای Hive:
```bash
flutter pub run build_runner build
```

4. اجرای برنامه:
```bash
# برای Windows
flutter run -d windows

# برای Web
flutter run -d chrome
```

## راهنمای استفاده

### 1. اضافه کردن دانش‌آموزان

1. به بخش "دانش‌آموزان" بروید
2. روی دکمه "بارگذاری Excel" کلیک کنید
3. فایل Excel حاوی لیست دانش‌آموزان را انتخاب کنید
   - فایل باید ستون "نام" داشته باشد
   - فرمت: .xlsx یا .xls

### 2. تکمیل کارنامه

1. از لیست دانش‌آموزان، روی "ویرایش کارنامه" کلیک کنید
2. به بخش "کارنامه" بروید
3. اطلاعات دانش‌آموز را تکمیل کنید:
   - مقطع، پایه، آموزشگاه، سرمربی
   - تعداد جلسات، جلسات حاضر، ردیف عملکرد
4. برای هر تکنیک، سطح عملکرد را انتخاب کنید
5. از دکمه‌های Previous/Next برای رفتن به دانش‌آموز بعدی استفاده کنید

### 3. خروجی گرفتن

1. به بخش "خروجی" بروید
2. فرمت خروجی را انتخاب کنید (PDF یا Excel)
3. نوع خروجی را مشخص کنید:
   - همه کارنامه‌ها
   - فقط کارنامه فعلی
4. مسیر ذخیره را انتخاب کنید
5. روی دکمه "خروجی" کلیک کنید

## Keyboard Shortcuts

### Navigation اصلی
- `Ctrl + 1`: بخش دانش‌آموزان
- `Ctrl + 2`: بخش کارنامه
- `Ctrl + 3`: بخش خروجی

### کارنامه
- `Arrow Left`: دانش‌آموز قبلی
- `Arrow Right`: دانش‌آموز بعدی
- `Ctrl + S`: ذخیره کارنامه
